# 🚚 DTL-RO: Dynamic Route Optimization Tool

## 📌 Overview
DTL-RO is a Python-based logistics optimization system designed to improve delivery efficiency by finding the shortest and most efficient routes for vehicles.

## 🎯 Objectives
- Minimize travel distance and cost
- Optimize vehicle routes
- Improve delivery efficiency

## ⚙️ Features
- Multi-vehicle route assignment
- Distance calculation using coordinates
- Route optimization using Nearest Neighbor algorithm

## 🛠️ Tech Stack
- Python
- Math Library

```bash
python main.py
